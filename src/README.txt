Welcome to EconXP!

Commands:
/econxp add <player> <amount>			- Adds <amount> to <player>'s exp balance.
/econxp subtract <player> <amount>		- Subtracts <amount> from <player>'s exp balance.
/econxp clear <player>					- Clears <player>'s exp.
/econxp balance <player>				- Lets you see <player>'s exp.
/econxp version							- Check the current version of EconXP.
